# Origin-Based Classification Philosophy

## Core Principle

**Sentiment is NOT the deciding factor for classification; Authenticity and Objectivity are.**

The system classifies reviews based on **ORIGIN** and **MOTIVATION**, not sentiment.

## Classification Logic

### üõë FRAUDULENT REVIEWS (FAKE)

Reviews whose origin or motivation is non-organic and deceptive. Classified as FAKE regardless of sentiment.

#### Origin/Motivation Triggers:
1. **Synthetic**: Generated by AI/LLM (detected by Model 2)
2. **Financial**: Paid, incentivized, or associated with compensation (detected by Model A - Fraud Detection)
3. **Automated**: Posted by bots or spam networks (detected by Model 1 + Patterns)
4. **Non-Objective**: Biased language patterns (detected by Model B - Bias Detection)

### ‚úÖ AUTHENTIC REVIEWS (REAL)

Reviews that originate from genuine, subjective human experience. Classified as REAL regardless of sentiment.

#### Origin/Motivation Triggers:
1. **Organic**: Written by a human user based on actual experience
2. **Objective Opinion**: Expressing genuine, un-incentivized feedback
   - Can be highly positive (great experience)
   - Can be highly negative (terrible experience)
   - Can be neutral (mixed experience)

## Decision Priority

1. **Model 2 (AI Detector)**: Primary authority on human vs. AI origin
   - If `model2.real_score > 0.6`: Likely human-written ‚Üí REAL
   - If `model2.fake_score > 0.4`: Likely AI-generated ‚Üí FAKE

2. **Bias/Fraud Detection**: Overrides everything if detected
   - Fraud detected ‚Üí FAKE (regardless of sentiment)
   - Bias detected ‚Üí FAKE (regardless of sentiment)

3. **Suspicious Patterns**: Supporting evidence, not primary
   - Only used if Model 2 is uncertain
   - Detects generic AI patterns (not sentiment)

4. **Model 1 (Review Classifier)**: Secondary signal
   - Combined with Model 2 for ensemble decision

## Key Changes

### Removed Sentiment-Based Logic
- ‚ùå No longer assumes negative reviews are more authentic
- ‚ùå No longer penalizes positive reviews as suspicious
- ‚úÖ Focuses on origin detection (human vs. AI vs. paid)

### Pattern Detection Focus
- Detects generic AI patterns (repetitive, lack of specifics)
- Detects excessive positivity ONLY if combined with AI indicators
- Does NOT penalize negative sentiment

### Model 2 Authority
- Model 2 (AI Detector) is the primary authority on origin
- If Model 2 says human-written, trust it (regardless of patterns)
- If Model 2 says AI-generated, classify as FAKE (regardless of sentiment)

## Implementation

All classification logic in `ensemble_classifier.py` now:
- Prioritizes Model 2 (AI Detector) as authority on origin
- Uses reasoning that mentions "origin" not "sentiment"
- Trusts Model 2's human detection over pattern-based suspicions
- Applies Bias/Fraud detection as override (non-objective = FAKE)
